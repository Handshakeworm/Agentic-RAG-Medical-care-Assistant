## 阈值统一(2026-05-05 user 拍板)

- `PARENT_SPLIT_THRESHOLD` = **5000** — section > 此值触发 Pass 1(单本可调,神经外科学/神经内科学/协和呼吸 6000)
- `PARENT_REFINE_THRESHOLD` / `PARENT_PASS3_THRESHOLD` / `PARENT_PASS4_THRESHOLD` = **6000** — Pass 1 后仍 > 6000 触发 Pass 2/3/4
- `PARENT_MERGE_TINY_THRESHOLD` = 500 — section 内二次切产物 < 500 字 + 同/上级相邻 → 合并
- `CHAPTER_ABSORB_THRESHOLD` = 500 — L1/L2/(L3) 短 section → 跨 section 吸收(L4 不参与)
- `CHILD_SPLIT_THRESHOLD` = 1200 — 父块 ≤ 1200 → 整块当 1 子块,不切
- `CHILD_TARGET_SIZE` = 600 / `CHILD_MIN_SIZE` = 200

每本书的统计数据格式跟 `python poc_chunk_book.py` 输出对齐(可直接 copy)。
