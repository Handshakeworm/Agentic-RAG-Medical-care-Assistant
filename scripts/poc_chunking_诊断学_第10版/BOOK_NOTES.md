## 3. 多页 TOC 处理(诊断学新发现的 mineru bug)

**问题**:目录跨 pg 15-21,但 mineru 只在 pg 15 加 `page_header="目录"`,后续 6 页没标识。

**应对**:`_detect_toc_pages` 启发式延伸:首页后,后续 page anchor 命中 ≥ 2 算延续,直到不命中为止。

```python
def _detect_toc_pages(data):
    seeds = [i for i, p in enumerate(data) if _is_toc_page(p)]
    extended = set(seeds)
    i = max(seeds) + 1
    while i < len(data):
        if _page_anchor_count(data[i]) >= 2:
            extended.add(i)
            i += 1
        else:
            break
    return sorted(extended)
```

| 其他 | 195 | 10% | "绪论" / "诊断学的学习内容" |

**层级关系(各节内,固定顺序)**:
```
节 (L0)
  ├─ 一、 (L1, CN_NUM)        ← 节内主题切换(疾病分组、检查项目分类等)
  │   ├─ 【...】 (L2, BRACE)  ← 主题下子方面(概述/症状/体征/诊断要点)
  │   │   └─ (一) (L3, PAREN) ← 子方面下子项
  │   │       └─ 1. (L4, NUM) ← 最细列表项
```

**注意**:不同节用不同的 anchor 子集 — 第一节 发热 用 【】+1./2.(无 一、);第一节 全身状态检查 用 一、+(一)+1./2.(无 【】)。
